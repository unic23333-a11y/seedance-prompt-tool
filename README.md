# Seedance 2.0 Prompt 工坊

**在线工具：** https://unic23333-a11y.github.io/seedance-prompt-tool/

---

## 是什么

一个针对 Seedance 2.0（字节跳动·豆包）的结构化提示词生成工具。

支持文生视频、图生视频、视频编辑、视频延长四种模式，提供官方案例模板和运镜速查表，一键复制生成结果。

---

## 核心功能

- **四种生成模式**：文生视频 / 图生视频 / 视频编辑 / 视频延长
- **结构化输入**：主体 → 动作 → 运镜 → 风格 → 约束
- **5 个官方案例**：点击即可填充完整示例
- **运镜速查表**：慢推、环绕、微距、延时等中英文对照
- **Negative Prompt 模板**：按场景（电商/UGC/电影感）提供防翻车约束

---

## 提示词公式

```
[风格定义] + [场景/主体] + [运镜] + [灯光/氛围] + [音效] + [约束]
```

### 关键原则

1. **一次只给一个主导运镜** — 不要同时要求 orbit + zoom + whip pan
2. **保护脆弱元素** — 手部/面部/标签等，必须在 Negative Prompt 中明确写出
3. **时间线分段** — 用「0～4秒」「5～12秒」精确控制节奏变化
4. **先窄后宽** — 结果不稳时，优先缩小任务再逐步加复杂度
